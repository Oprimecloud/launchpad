## ğŸ–Œ How to Customize

1. **Change Text**
   - Open `index.html`
   - Replace headings, paragraphs, and button text with your own content

2. **Replace Images**
   - Navigate to `assets/img/`
   - Replace `hero.jpg`, `logo.png`, and other placeholders with your own images
   - Keep file names or update the HTML `src` attributes accordingly

3. **Change Colors**
   - Open `assets/css/style.css`
   - Update color values (`#e60023`) to match your brand palette

4. **Customize Buttons / Links**
   - Buttons use `.btn-primary` or `.btn-secondary` classes
   - Update href attributes to your links or forms

5. **Add JS Interactions (Optional)**
   - Edit `assets/js/main.js` to add animations or interactivity
